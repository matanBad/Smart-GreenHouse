5. Automation APIs

Create the following APIs:

GET /api/actuators

Returns all actuators and their current status.

GET /api/automation/actions

Returns all automation actions ordered from newest to oldest.

GET /api/automation/active

Returns currently active actuator responses.

PATCH /api/actuators/<actuator_id>/reset

Resets selected actuator status to inactive.

- Reset action should be logged.
- Resetting inactive actuator should return a safe message.
